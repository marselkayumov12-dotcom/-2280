## 🔑 How to Get an API Key

This app uses **ExchangeRate API** (free tier available):

1. Go to [https://www.exchangerate-api.com/](https://www.exchangerate-api.com/)
2. Click **"Get Free API Key"**
3. Enter your email and complete registration
4. You'll receive a 24-character API key via email
5. In the code (`currency_converter.py`), replace:
   ```python
   self.api_key = "YOUR_API_KEY"
